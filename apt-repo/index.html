üîß Manual Repository Setup

To manually add this repository to your system:

# Add TulioCP repository
echo "deb https://apt.tuliocp.com stable main" | sudo tee /etc/apt/sources.list.d/tuliocp.list

# Update package lists
sudo apt-get update

# Install TulioCP
sudo apt-get install hestia tulio-nginx tulio-php tulio-web-terminal
